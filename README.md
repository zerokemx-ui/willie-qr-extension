# Willie QR 產生器 | Stylish QR Maker

一鍵把網址或文字變成漂亮的 QR Code 的 Chrome 擴充功能。支援圓點、圓角、正圓樣式，可加入中心 LOGO，輸出 PNG 或 SVG 向量圖。

## 功能

- 即時預覽：輸入文字或網址，QR Code 立刻產生
- 多種資料點樣式：標準方塊、圓點、圓角方塊、間隔方塊
- 定位點設計：外框與內芯可獨立選擇直角、圓角、正圓
- 中心 LOGO：上傳品牌 Logo，自動調整容錯率保證可掃描
- 一鍵抓取：自動填入目前分頁的網址
- 向量輸出：PNG 點陣圖或 SVG 向量圖
- 複製到剪貼簿、深色模式、UTF-8 完整支援

## 隱私

所有運算 100% 在你的瀏覽器本機完成，不會上傳任何內容到伺服器。詳見 [PRIVACY.md](PRIVACY.md)。

## 本機開發 / 安裝

1. 開啟 `chrome://extensions/`
2. 右上角啟用「開發人員模式」
3. 點「載入未封裝項目」→ 選這個資料夾
